Valor dos inputs -> .value

Conteúdo do HTML de uma tag -> .innerHTML -> Processa o código HTML

Conteúdo textual de uma tag -> .textContent -> Não processa os códigos

Manipulação de atributos -> .setAttribute('atributo','valor')

Acessar atributos -> .getAttribute('atributos')

remove atributos -> .removeAttribute('atributos')

Manipulação de Estilo

Modificação de estilos

  • Adicionar -> .classlist.add('Classe')
  • Remover -> .classlist.remove('Classe')
  • Trocar -> .classlist.toggle('Classe')
  • Verificar classe -> .classlist.contains ('Classe')
